Getting started

TapeReplay is a desktop app (Electron) with a local .NET API and React UI. You configure strategies, run backtests against real minute bars in SQLite, and visualize trades on candlestick charts.

What you need

  • Installed app — DMG/NSIS from a release, or run from source with make dev.
  • Backend running — the green dot in the header means the API on port 5180 is healthy.
  • Minute bar data — either mock data (default) or your own SQLite cache from Polygon recording.

First launch checklist

  1. Wait for “Backend: Connected” — on first launch the .NET API may take a few seconds. If it stays amber, restart the app or run make dev from a terminal to see errors.
  2. Try mock mode — out of the box, UseMockProvider: true serves synthetic bars so you can click Exploratory Day without a Polygon key.
  3. Read the honesty rules — a single green day is not proof. See Honesty by design before trusting any number.
  4. Record real data — when ready, add a Polygon key and run make record. See Collecting data.
  5. Visualize a day — open Chart backtest, pick a ticker and date with stored bars, run ORB or PMH, and compare to perfect hindsight.
Tip: Use the header tabs to switch between Strategy lab (train/test workflow) and Chart backtest (intraday candlesticks with trade markers).

Development quickstart

From the repo root:

make install   # first time
make dev       # API + Vite + Electron

Polygon API key (never commit): copy backend/appsettings.Development.local.json.example to backend/appsettings.Development.local.json and set your key.
